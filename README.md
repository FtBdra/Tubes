# Analisis Perbandingan Kompleksitas Algoritma Iteratif dan Rekursif untuk Memverifikasi Palindrom pada Kalimat
Mengimplementasikan algoritma **iteratif** dan **rekursif** melalui aplikasi **Streamlit** untuk memverifikasi apakah sebuah kalimat merupakan palindrom. Dengan menganalisis kompleksitas waktu (membandingkan running time) dan perbedaan efisiensi yang signifikan pada penggunaan memori.

Aplikasi melakukan:
- Normalisasi teks dari input pengguna (menghapus spasi, simbol, dan kapital) menjadi karakter yang seragam.
- Verifikasi palindrom menggunakan dua pendekatan:
  - **Iteratif**
  - **Rekursif**
- Analisis **waktu eksekusi (running time)**
- Analisis **penggunaan memori**
- Visualisasi hasil menggunakan grafik batang

---
## Menggunakan Fitur dan Library
- Python
- Streamlit
- Matplotlib
- Tracemalloc

### Cara menjalankan program di komputer pribadi
1. Install Fitur dan Library

   ```
   $ pip install -r requirements.txt
   ```
2. Jalankan kode dengan

   ```
   $ streamlit run palindrom.py
   ```
